Awesome KivyMD
==============

.. image::https://github.com/quitegreensky/akivymd/blob/master/images/preview.gif


Awesome KivyMD is a collection of non material widgets built using Kivy and KivyMD widgets.
The project's goal is to create a collection of beautifully designed widgets that can be easily used within your kivy programs.
The project is constantly evolving and new widgets are contstantly being added.
If you wish to contribute to this project just make a pull request over at our GitHub repository.
